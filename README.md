# 群友 CP 随机抽取插件

一个为 AstrBot 框架设计的 QQ 群聊娱乐插件，支持随机抽取群成员作为"老公"/"老婆"，生成关系图等功能。

## 功能特性

- **随机老公/老婆**：从群成员中随机抽取一位作为你的 CP
- **换老公/换老婆**：支持重新抽取，每天更换次数可配置（默认各 1 次）
- **绑定模式**：开启后双向绑定，A 抽 B 为老公，则 B 抽老婆必为 A
- **关系图**：生成群友 CP 关系可视化图表，红色箭头表示老婆关系，蓝色箭头表示老公关系
- **每日重置**：每天 00:00 自动重置所有抽取结果和更换次数
- **数据持久化**：抽取结果保存到本地，重启 Bot 不丢失
- **多群独立**：每个群的数据完全独立

## 支持平台

仅支持 **aiocqhttp**（QQ 个人号 / NapCat / LLOneBot 等 OneBot 11 实现）。

## 安装方法

1. 将本插件文件夹复制到 AstrBot 的插件目录：
   ```bash
   cd /AstrBot/data/plugins
   git clone https://github.com/mangokuma/astrbot_plugin_cp_random
   ```

2. 在 AstrBot WebUI 的「插件管理」中重载插件。

3. 首次使用时会自动安装 `Pillow` 和 `aiohttp` 依赖。

## 使用指令

### 核心指令

| 指令 | 别名 | 说明 |
|------|------|------|
| `随机老公` | `今日老公` | 从群友中随机抽取一位作为你的老公 |
| `随机老婆` | `今日老婆` | 从群友中随机抽取一位作为你的老婆 |
| `换老公` | — | 重新随机抽取一位老公，消耗更换次数 |
| `换老婆` | — | 重新随机抽取一位老婆，消耗更换次数 |
| `关系图` | — | 生成当前群的 CP 关系图（圆形头像 + 红蓝箭头） |

### 管理指令

| 指令 | 说明 | 示例 |
|------|------|------|
| `CP设置 模式 <bind/random>` | 设置抽取模式（仅群主/管理员） | `CP设置 模式 bind` |
| `CP设置 次数 <1-10>` | 设置每天更换次数（仅群主/管理员） | `CP设置 次数 3` |
| `CP设置 状态` | 查看当前群的设置状态（仅群主/管理员） | `CP设置 状态` |

### 模式说明

- **非绑定模式**（默认）：每次抽取完全随机，互不影响。
- **绑定模式**：A 使用「随机老公」抽到 B，则 B 使用「随机老婆」时必然抽到 A。但 B 使用「随机老公」仍然随机抽取。

## 消息格式

抽取成功后会发送：
- 文本：恭喜 xxx，你的老公/老婆是 xxx（QQ号）~
- 图片：被抽取者的 QQ 头像

如果头像获取失败，会只发送文本消息。

## 注意事项

1. 机器人需要在群内有获取群成员列表的权限。
2. 更换次数每天 00:00 自动重置。
3. 数据存储在 `data/plugins/astrbot_plugin_cp_random/cp_data.json`。
4. 关系图生成需要下载头像，群成员较多时可能需要几秒钟。
5. 关系图最大分辨率限制为 1024×1024，无中文字体时昵称自动隐藏。
